# Instruções de Uso e Documentação

Este projeto tem como objetivo fornecer uma forma eficiente de inserir dados.

## Requisitos
- Certifique-se de ter o Python 3.x instalado em seu sistema.
- Dependências necessárias:
  - pandas
  - numpy

## Instalação
Para instalar as dependências, você pode usar o pip:
```bash
pip install -r requirements.txt
```

## Como Usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/yungGenos/Isert_dados.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd Isert_dados
   ```
3. Execute o script principal:
   ```bash
   python main.py
   ```

## Estrutura do Código
- `main.py`: Script principal que executa a inserção de dados.
- Outros módulos com utilidades e funções auxiliares.
